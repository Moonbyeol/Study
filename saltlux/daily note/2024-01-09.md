### 논리 연산자
- 조건들을 결합하거나 반전시켜 더 복잡한 조건을 만들 수 있음
- ```not```: 논리값을 반전시킴(True -> False, False -> True)
- ```and```: 모든 조건이 참일 때 True 반환
- ```or```: 조건 중 하나라도 참이면 True 반환
