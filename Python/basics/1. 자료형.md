
## 1. 숫자형

- 정수형 int
- 실수형 float
### 사칙 연산
``` python
a = 7
b = 4

# 더하기
a + b = 11
# 빼기
a - b = 3
# 곱하기
a * b = 28
# 나누기
a / b = 1.75
# 제곱
a ** b = 2401
# 몫
b // a = 1
# 나머지
b % a = 3
```


## 2. 문자열 자료형

```python

|n : 문자열 안에서 줄을 바꿀 때 사용
|t : 문자열 사이에 탭 간격을 줄 때 사용
|| : |를 그대로 표현할 때 사용
|' : 작음따옴표(')를 그대로 표현할 때 사용
|" : 큰따옴표(")를 그대로 표현할 때 사용
|r : 캐리지 리턴(줄 바꿈 문자, 커서를 현재 줄의 가장 앞으로 이동)

```
