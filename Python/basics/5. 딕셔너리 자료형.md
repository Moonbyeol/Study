### 딕셔너리 추가
```python
a = {1: 'a'} 
a[2] = 'b' 
a = {1: 'a', 2: 'b'}

a['name'] = 'pey'
a = {1: 'a', 2: 'b', 'name': 'pey'}

a[3] = [1, 2, 3]
a = {1: 'a', 2: 'b', 'name': 'pey', 3: [1, 2, 3]}

```

### 딕셔너리 삭제
```python
del a [1]
a = {2: 'b', 'name': 'pey', 3: [1, 2, 3]}
```

### 딕셔너리에서 Key를 사용해 Value 얻기
```python
dic = {'name':'pey', 'phone':'010-9999-1234', 'birth': '1118'} >>> dic['name'] = 'pey'
dic['phone'] = '010-9999-1234'
dic['birth'] = '1118'
```
