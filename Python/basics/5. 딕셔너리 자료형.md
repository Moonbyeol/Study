```python
{Key1: Value1, Key2: Value2, Key3: Value3, ...}
```


### 딕셔너리 추가
```python
a = {1: 'a'} 
a[2] = 'b' 
a = {1: 'a', 2: 'b'}

a['name'] = 'pey'
a = {1: 'a', 2: 'b', 'name': 'pey'}

a[3] = [1, 2, 3]
a = {1: 'a', 2: 'b', 'name': 'pey', 3: [1, 2, 3]}

```

### 딕셔너리 삭제
```python
del a [1]
a = {2: 'b', 'name': 'pey', 3: [1, 2, 3]}
```
