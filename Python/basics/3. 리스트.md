### 리스트 연산하기
```python
a = [1,2,3]
b = [4,5,6]

a + b = [1,2,3,4,5,6] # 더하기
a * 3 = [1,2,3,1,2,3,1,2,3] # 반복하기
len(a) = 3  길이 구하기
str() : 정수나 실수를 문자열로 변환
int() : 정수형 으로 변환
del a[1] : a = [1,3] 리스트 요소 삭제
```

### 리스트 함수
```python
.append() : 리스트에 요소 추가하기
.sort() : 리스트 정렬
.reverse() : 거꾸로 뒤집기
.index() : 위치 찾기
.insert(a,b) : a위치에 b 삽입
.remove(x) : 리스트에서 첫 번째로 나오는 x를 삭제
.pop() : 리스트의 맨 마지막 요소를 리턴하고 그 요소를 삭제
.pop(x) : 리스트의 x번쨰 요소를 리턴 하고 삭제
.count(a) : 리스트 안에 a가 몇 개 있는지 개수 세기
.extend(x) : x에는 리스트만 올 수 있으며 원래 리스트에 x 리스를 더한다
```
